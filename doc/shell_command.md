## shellコマンドリスト

|            やりたい事              |    コマンド例     |
|  :------------------------------: | :--------------: |
|  別の場所へ移動                    |  cd ./workspace  |
|  1つ上のディレクトリへ移動          | cd .．  |
|  rootディレクトリへ移動            | cd / |
|  カレントディレクトリの確認         | pwd |
|  ファイル表示                      | ls |
|  隠しフォルダ(ファイル)も表示       | ls -a |
|  新規フォルダを作る                | mkdir Sample |
|  フォルダを消す                    | rm -rf Sample |
|  新規ファイルを作る                | touch sample.txt |
|  ファイルを消す                    | rm sample.txt |
|  コピー(ファイル/フォルダどちらも可) | cp ./README.md(コピー元) C:/Users/gn5r/Desktop(コピー先) |
